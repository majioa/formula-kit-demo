Введение
--------

Библиотека классов Formula.lib - это библиотека для 
Borland Delphi и Borland C++ Builder, включающая исходные тексты 
всех модулей и демонстрационный пример.

Библиотека содержит головной класс TFormulaKit, предназначенный 
для совместной обработки группы формул. Формулы представляют собой 
именованные арифметические выражения. Имена формул могут входить в 
другие формулы в качестве переменных. При любом изменении 
содержимого набора формул происходит пересчет всех значений.

Библиотеа представляет набор типовых операций по модификации набора 
формул и извлечению значений.

Использование
-------------

Для использования библиотеки достаточно добавить ее к проекту и 
подключить к модулям проекта заголовок TFmlaKit.h.
Проект должен иметь доступ к шести файлам заголовков библиотеки,
а также к заголовку Parsing.hpp из библиотеки RX.

По отношению к объекту класса TFormulaKit возможны следующие действия:

- добавить формулу с именем Name, значением Mean 
	AddFormula(String Name, String Mean)
	Имя может содержать буквы латинского алфавита и цифры и должно 
	начинаться с буквы.

- изменить значение ранее добавленной формулы с именем Name на
	новое значение Mean 
	Change(String Name, String Mean)

- удалить формулу с именем Name
	Delete(String Name)

- удалить все формулы
	Clear()

Любое из вышеприведенных действий вызывает пересчет значений всех 
формул набора. Следующие методы позволяют исследовать содержимое
набора формул и получать результаты расчета

- возвратить выражение формулы с именем Name 
	String GetMean(String Name);

- возможен ли расчет значения формулы с именем Name 
	bool CanGet(String Name);

- возвратить значение формулы  с именем Name
	double __fastcall Get(String Name)

Демонстрационные примеры
------------------------

Демонстрационная программа FmlaDemo.exe, входящая в состав 
библиотеки, предназначена для демонстрации работы библиотеки.

Программа демонстрирует создание и вызов основных методов класса 
TFormulaKit. Результаты отображаются в сетке. Формулы, которые не 
могут быть вычислены, имеют в сетке пустые клетки значений.

Исходные тексты
---------------

Библиотека включает шесть модулей, содержащие шесть классов двух
линий наследования TVariable-TExpression-TFormula и 
TVariableKit-TExpressionKit-TFormulaKit.

Все модули библиотеки поставляются с исходными
текстами и доступны для изучения, доработки, изменения и т.д.

Примечания
----------

Библиотека использует математический анализатор из библиотеки 
компонентов RX.        

Библиотека не поддерживает экспоненциальный формат записи чисел,
например: -2.056e-02 следует заменить на -0.02056.

Кроме метода добавления формулы AddFormula класс TFormulaKit имеет
также методы добавления выражения AddExpression и переменной AddVariable 
вызов которых аналогичен вызову AddFormula. Отличия объектов, добавленных
как выражение, заключается в том, что они не могут ссылаться на другие
объекты (формулы, выражения и переменные). Переменные не могут 
также содержать арифметические выражения и функции, а только числа.
